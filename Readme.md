# git-step

CLI tool to step through git commits in either direction


## Why 

Useful for live tutorials, talks, etc.

## Install

```sh
npm i -g git-step
```

This will install `git-next` and `git-prev` executables, and
automatically alias them to `git next` and `git prev`.

## Usage

In a git repo

```sh
git prev <steps>
git next <steps>
```
